# Video Standards

This folder defines the standardised approach for creating product demo videos within Demo Media.

The goal is to reduce variation, improve reuse, and ensure all content follows a consistent structure, tone, and production process.

---

## What’s included

- [**workflow.md**](workflow.md)
  Defines the end-to-end production process from research to final delivery.
  
- [**AI-script-standard.md**](AI-script-standard.md)
  Defines how scripts should be written, structured, and refined using AI.

- [**imagery-generation-standard.md**](imagery-generation-standard.md)
  Covers how to create images, assets, and AI-generated video reliably and consistently.

---

## How to use

1. Start with [**workflow.md**](workflow.md) to understand the process.
2. Use [**AI-script-standard.md**](AI-script-standard.md) to write and refine scripts.
3. Use [**imagery-generation-standard.md**](imagery-generation-standard.md) when creating visuals and animations.
4. Follow review checkpoints at every stage.
